Class: RSLPStemmer

RSLPStemmer()

Implementation fo the algorithm for the RSLP Stemmer, as described in the "A Stemming Algorithm for the Portuguese Language" paper.

In Proceedings of the SPIRE conference, Laguna de Sal Raphael, Chile, November 13-15, 2001, written by Viviane Moreira Orengo and Christian Huyck

More info: http://www.inf.ufrgs.br/~viviane/rslp/index.htm Datasets from: https://www.kaggle.com/nltkdata/rslp-stemmer Source: https://www.andrewsaguiar.com/blog/2019/09/12/text-search-implementing-portuguese-stemmer
